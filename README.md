# Email Images

This repository hosts images for use in emails and other HTML content.

## How to Use Images

To use these images in your HTML, use the GitHub raw content URL format:

```
https://raw.githubusercontent.com/chris853783/email-imges/main/<filename>
```

### Example HTML img Tags

**file-1762794941514.png:**
```html
<img src="https://raw.githubusercontent.com/chris853783/email-imges/main/file-1762794941514.png" alt="Image">
```
